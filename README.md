# Think-Next-Assignment
Think Next Assignment
Court Case Scraper & PDF Generator

Fetch court case details from eCourts India
 and save them as PDFs or JSON. Search by CNR number or case type easily.

Features

- Fetch case details by CNR or Case Type / Number / Year

- Generate PDF reports of case listings

- Extract case info from existing PDFs

- Save results as JSON for further use

- Print upcoming hearings (today/tomorrow)
